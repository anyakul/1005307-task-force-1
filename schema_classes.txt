Классы. Схема mvc

ШАБЛОНЫ.
1) layouts
- layout - теги <head> <footer> с содержанием, $headerContent, $mainContent, $modal, $script, переменные для вставки header и main, скриптов и модальных окон.
- LandingPageHeader - хедер лэндинг страницы
- MainPageHeader - хедер страниц авторизированного пользователя

2) LandingPage - содержание лэндинга со вставкой последних заданий
- LandingPage - основное содержание
- SignUpModal - модальное окно входа на сайт
- Script - подключение js

3) Tasks -
- NewTasksPage - список новых заданий с фильтрами и поиском
- CreateTaskPage - страница создания нового задания
- MyTasksPage - страница моих заданий
- ViewTaskPage - страница просмотра задания - инфа о задании, отклики, чат
- Script - подключение js
- Modal - модальные окна на страницах заданий

4) Users
- UsersPage - страница списка исполнителей с фильтами
- ViewUsersPage - страница с инфой о пользователе - с отзывами

5) Account
- RegistrationPage - страница регистрации пользователя
- SettingPage - страница настроек пользователя

КОНТРОЛЛЕРЫ
- AccountController - работа аккаунтом пользователя
- LandingController - работа с лэндингом
- LogoutController - выход пользователя
- SignUpController - регистрация пользователя
- TaskController - работа с заданиями
- UserController - работа с пользователями
- MessageController - работа с сообщениями
- OpinionsController - работа с отзывами

MOДЕЛИ для таблиц базы данных

папка forms
- LoginForm - форма авторизации
- SignupForm - форма регистрации
- TaskForm - форма создания задания
- MessageForm - форма отправки сообщения
- ReviewForm - форма отправки отклика на задание
- UserForm - форма отправки отзыва

вне папок:
- Categories  категории
- Cities города
- Favourites - избанные
- FileTask - файлы заданий
- Message - сообщения
- Notifications - уведомления
- Opinions - отзывы
- PortfolioPhoto - фото портфолио
- Response - отклики
- Task - задания
- User - пользователи
- UserCategory - пользователей с категориями
