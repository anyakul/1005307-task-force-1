Классы. Схема mvc

Шаблоны. содержит html-код - шаблоны, вставка php-переменных
- AbstractTemplate - содержит метод вывода html кода на страницу (include_template)
- LandingPage - главная страница сайта
- RegistrationPage страница регистрации пользователя
- AutorisationPage - страница авторизации пользователя
- UsersPage - страница списка исполнителей
- UserPage - страница информации о пользователе
- SettingsPage - страница настроек пользователя
- TasksPage - страница всех новых заданий
- CreateTaskPage - страница создания нового задания
- TaskPage - страница информации о задании
- MyTasksPage - страница моих заданий
- Layout - header и footer страницы
- Announces - окно уведомления

модели
- Api - методы создания бд, добавления, редактирования, удаления из базы данных, обработка ошибок (mysqlconnect...), возможно как абстрактный класс...
- User - инфа о конкретном пользователе, как в базе данных + методы редактирования инфы о пользователе
- Users - методы получения обработки списка пользователей в соответствии с установленными фильтрами
- Task - инфа о конкретном задании, как в базе данных + методы редактирования инфы о задании
- Tasks - методы получения обработки списка заданий в соответствии с установленными фильтрами
- Message - инфа о конкретном сообщении, как в базе данных + методы отправки
- Messages - методы получения обработки списка сообщения
- Review - инфа о конкретном отзыве, как в базе данных + методы редактирования
- Reviews - методы получения обработки списка отзывов
- Response - инфа о конкретном отклике на задание, как в базе данных + методы редактирования
- Responses - методы получения обработки списка откликов
- PhotoWork - фото работы пользователя, как в базе данных + методы редактирования
- PhotoWorks - методы получения обработки списка фотографий выполненных работ
- FileTask - инфа о файлах задания как в базе данных + методы редактирования
- FileTasks - методы получения обработки списка файлов конкретного задания
- Announce - инфа о конкретном уведомлении, как в базе данных + методы уведомлений
- Announces - методы получения обработки списка уведомлений

controllers
- ConnectMySql - связь с базой данных
- Error - обработка ошибок
- FormValidate - валидация форм
- AbstractController - методы рендеринга шаблонов
- RenderUsers - связь шаблонов и моделей, связанных с пользователями (регистрация, авторизация, настройки, вывод списка пользователей, их фильтр...)
- RenderTasks - связь шаблонов и моделей, связанных с заданиями (редактирование и создание заданий, вывод списка заданий, фильтр заданий соответствии с моделями...)
- RenderMessages - связь шаблонов и моделей, связанных с перепиской между пользователями по заданию
- RenderOPinions - связь шаблонов и моделей, связанных с отзывами о пользователе
- RenderResponses - связь шаблонов и моделей, связанных с откликами на задание
